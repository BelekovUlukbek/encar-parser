# Парсер Encar - Экспериментальный проект

## 📋 Описание
Проект для сбора данных с Encar.com и создания базы корейских автомобилей.

## 🚀 Быстрый старт

### 1. Установка
```bash
# Клонировать или создать папку
mkdir encar-parser
cd encar-parser

# Создать виртуальное окружение (рекомендуется)
python -m venv venv

# Активировать
# Windows: venv\Scripts\activate
# Mac/Linux: source venv/bin/activate

# Установить зависимости
pip install -r requirements.txt
